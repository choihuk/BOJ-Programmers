## 무식하게 구하기
```java
 public boolean isPrime(int sum) {
      for(int i=2; i<sum; i++) {
          if (sum%i == 0) {
              return true;
          }
      }
      return false;
  }
```

## 똘똘하게 구하기
```java
  public boolean isPrime(int num){
      for(int i = 2; i <= (int)Math.sqrt(num); i++){
        if(num % i == 0) {
            return false
        }
      }
      return true
  }
```
